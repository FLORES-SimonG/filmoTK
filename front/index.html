Filmo-TK

¡Hola Mundo!

  • La página web "Filmo-TK", creada por Simón Flores, es una web orientada al ámbito cinematográfico.
  • En la sección "Formulario" de la web, es posible participar en la creación de películas.
  • Este proyecto forma parte de los requisitos para rendir el exámen del módulo n°2 de SoyHenry, donde tras la aprobación de éste permite acceder al exámen teórico del módulo.
  • En términos técnicos, "Filmo-TK" ha sido desarrollada utilizando CSS y Bootstrap para el diseño Front, ofreciendo una experiencia dinámica. Para el backend, se ha empleado por tecnologías como Mongo y Express, sumando también otras extensiones vistas durante el cursado.
  • Este proyecto intenta en reflejar las habilidades técnicas adquiridas a lo largo de lo aprendido durante el Bootcamp.
